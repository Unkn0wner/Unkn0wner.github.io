## 📋 STEP-BY-STEP DEPLOYMENT GUIDE

### STEP 1: Create GitHub Account (2 minutes)
1. Open a new browser tab
2. Go to: **github.com**
3. Click the green **"Sign up"** button
4. Enter your email (use your student email if you have one!)
5. Create a password
6. Choose a username - suggestions:
   - `esdesignexcellency` 
   - `esdesign-portfolio`
   - `your-first-name-design`
7. Verify your email
8. Choose the FREE plan (it's unlimited!)

✅ **You now have a GitHub account!**

---

### STEP 2: Create Your Website Repository (3 minutes)

1. Once logged into GitHub, click the **+** icon (top right corner)
2. Select **"New repository"**
3. **IMPORTANT:** Name your repository exactly like this:
   ```
   YOUR-USERNAME.github.io
   ```
   Example: If your username is `esdesignexcellency`, name it:
   ```
   esdesignexcellency.github.io
   ```
   
4. Make sure it's set to **Public** (required for free hosting)
5. ❌ **Don't** check "Add a README file"
6. Click **"Create repository"**

✅ **Your repository is created!**

---

### STEP 3: Upload Your Portfolio Files (3 minutes)

You'll see a page with several options. Follow this:

1. Look for the text that says **"uploading an existing file"** and click it
2. You'll see a drag-and-drop area
3. **Download all files from this folder** (I'll provide a zip)
4. Drag ALL files into the upload area:
   - `index.html`
   - All the `.jpg` image files
   - `README.md`
   - `.gitignore`
5. Scroll down to the bottom
6. In the "Commit changes" box, you can leave the default message
7. Click the green **"Commit changes"** button

✅ **Your files are uploaded!**

---

### STEP 4: Enable GitHub Pages (2 minutes)

1. On your repository page, click **"Settings"** (top menu)
2. In the left sidebar, scroll down and click **"Pages"**
3. Under **"Source"**, select **"Deploy from a branch"**
4. Under **"Branch"**, select **main** (or master)
5. Leave the folder as **/ (root)**
6. Click **"Save"**

You'll see a message: "Your site is ready to be published at..."

✅ **GitHub Pages is now building your site!**

---

### STEP 5: Visit Your Live Website! 🎉

**Wait 2-5 minutes** for the first build to complete.

Your portfolio will be live at:
```
https://YOUR-USERNAME.github.io
```

Example:
```
https://esdesignexcellency.github.io
```

## 🔄 HOW TO UPDATE YOUR SITE LATER

Whenever you want to update your portfolio:

1. Go to your repository on GitHub
2. Click on the file you want to edit (like `index.html`)
3. Click the **pencil icon** (top right) to edit
4. Make your changes
5. Scroll down and click **"Commit changes"**
6. Your site will automatically update in 1-2 minutes!

## 🎨 OPTIONAL: Add a Custom Domain

Want `www.yourname.com` instead of `username.github.io`?

1. Buy a domain from Namecheap, Google Domains, or GoDaddy (~$10/year)
2. In GitHub Settings → Pages, add your custom domain
3. Update your domain's DNS settings (I can help with this later!)

## ❌ TROUBLESHOOTING

**Problem:** Site shows 404 error
- **Solution:** Make sure your repository is named exactly `username.github.io`
- **Solution:** Check that `index.html` is in the root folder (not in a subfolder)

**Problem:** Images aren't showing
- **Solution:** Check that all image filenames match exactly (case-sensitive!)

**Problem:** Page not updating after changes
- **Solution:** Wait 2-3 minutes, then hard refresh (Ctrl+Shift+R on Windows, Cmd+Shift+R on Mac)
